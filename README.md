A website that allows you to export your ÄŒSFD film ratings.

You can find it at [csfd-export.ooooo.page](https://csfd-export.ooooo.page).

## Installation

### Mac

```shell
$ brew install poetry
$ make setup
```

### Arch Linux

```shell
# pacman -S python-poetry
$ make setup
```

### Other systems

Install these dependencies manually:

- Python >= 3.11
- poetry

Then run:

```shell
$ make setup
```

## Usage

Start a development server

```shell
$ make run
```
